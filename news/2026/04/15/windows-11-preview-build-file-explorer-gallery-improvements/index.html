Windows 11 Preview Build Brings Gallery View to File Explorer

By OfficeandWin Tech Desk · 2026-04-15

⚡ Quick Summary

  • Gallery view added to File Explorer in latest Windows 11 Insider build
  • Shows Photos app library directly in file browser
  • Supports OneDrive Camera Roll and custom folder filtering
  • Also available in file-picker dialogs for easy attachment
  • Rolling out gradually to Dev Channel Insiders

Microsoft Tests New Photo-Focused File Explorer Tab

Microsoft is making it easier to browse photos directly inside File Explorer. The latest Windows 11 Insider preview build introduces a dedicated Gallery view that surfaces the Photos app’s image library right inside the file browser—no app switching required.

What Is Gallery in File Explorer?

The new Gallery shortcut appears in File Explorer’s left navigation pane. Click it and the main browsing area switches from a traditional folder view to a thumbnail grid showing the same images you’d see in the Photos app’s “All Photos” view. If you have OneDrive Camera Roll Backup enabled on your phone, your most recent mobile photos appear at the top.

You can filter by folder through the Collection dropdown—picking desktop backgrounds, SD card imports, or any custom subfolder. Gallery also shows up as an option in the standard file-picker dialog, making it simpler to attach photos to emails, PowerPoint decks, or social posts.

Why It Matters for Business Users

For anyone who regularly inserts images into documents or presentations, this removes a friction point. Instead of navigating deep folder trees to find a screenshot or camera photo, you browse a chronologically sorted gallery. IT teams managing shared workstations will also appreciate that the Gallery view respects OneDrive folder policies—no local copies needed.

Current Availability

Gallery is rolling out gradually to Dev Channel Insiders. Microsoft says it will monitor feedback before expanding availability. As with all Insider features, there’s no guaranteed ship date for the stable channel, but the feature is polished enough that it’s likely to appear in a future Windows 11 feature update.

How to Try It

Join the Windows Insider Dev Channel, install the latest build, and look for Gallery in File Explorer’s sidebar. If you don’t see it yet, check back in a week—Microsoft rolls features out incrementally.

Frequently Asked Questions

When will Gallery in File Explorer be available to all users?

Microsoft is rolling it out gradually to Insiders first. Expect it in a stable Windows 11 update within 3-6 months.

Does Gallery replace the Photos app?

No, it surfaces the same photo library inside File Explorer. The Photos app remains available as a standalone experience.
